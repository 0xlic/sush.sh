# sush.sh (sush) Roadmap

- [x] 自动导入 ~/.ssh/config
- [x] 主界面搜索 + 主机列表
- [x] SSH 接管模式连接
- [x] Ctrl-Space 切换 SSH ↔ SFTP
- [x] 单面板 SFTP 文件浏览器
- [x] 文件上传/下载 + 进度显示
- [x] 底部快捷键提示栏
- [x] 双 Ctrl-C 返回主界面

## 长期愿景（v1.x+）

以下是差异化方向，不承诺时间线，按价值优先级排列：

### 会话录制回放
- asciinema 格式录制 SSH 会话
- 终端内回放，支持搜索
- 用途：运维审计、教学演示、故障复盘

### 智能文件同步
- 本地目录 ↔ 远程目录的 watch + auto-sync
- 类似 VS Code Remote 的开发体验但在终端里
- .gitignore 规则排除
- 冲突检测

### Mosh 模式
- 弱网环境下的 UDP 连接
- 本地即时回显
- 需要远程安装 mosh-server，或自实现 SSP 协议子集

### 跨主机文件传输
- 选中两台远程主机，直接在它们之间传输文件
- 通过本地中转或 SSH 隧道直传

### 插件系统
- 自定义连接后自动执行的脚本
- 自定义快捷键绑定
- 主题自定义
