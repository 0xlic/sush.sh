# sush.sh (sush) Roadmap

## v0.1 — MVP（已完成）

核心体验：搜索主机 → SSH 连接 → 无缝切 SFTP → 上传下载文件

- [x] 自动导入 ~/.ssh/config
- [x] 主界面搜索 + 主机列表
- [x] SSH 接管模式连接
- [x] Ctrl-\ 切换 SSH ↔ SFTP
- [x] 单面板 SFTP 文件浏览器
- [x] 文件上传/下载 + 进度显示
- [x] 底部快捷键提示栏
- [x] 双 Ctrl-C 返回主界面

## v0.2 — 嵌入式终端模式（已完成）

**架构调整**：将 SSH 模式从接管模式（LeaveAlternateScreen + I/O 直传）切换为嵌入式终端模拟器模式，sush 界面在 SSH 连接期间始终可见。

- [x] 集成 `alacritty_terminal` 作为 VT100/xterm 状态机
- [x] 实现 `TerminalView` ratatui widget（将虚拟屏幕 cell 渲染到 ratatui 画布）
- [x] 状态栏在 SSH 模式下持续显示主机名与快捷键提示
- [x] 前缀键（Ctrl-\）在 TUI 层处理，不再需要切换 alternate screen

## v0.3 — 主机管理 CRUD（已完成）

- [x] TUI 内新建/编辑/删除主机
- [x] 标签管理（增删改、按标签筛选）
- [x] 主机描述字段展示与编辑
- [x] SSH config 手动导入（取代自动合并）
- [x] 首次启动引导导入流程

## v0.4 — 使用体验增强（已完成）

- [x] 连接历史记录（`history.toml`）+ 最近使用倒序排列
- [x] 模糊搜索中近期连接主机加权排名靠前
- [x] 主机连通性检测：悬停 1 秒后自动 TCP probe，描述行 ● 显示绿/红/灰状态
- [x] 连接错误在描述行显示（红字），4 秒后自动恢复

## v0.5 — 虚拟目录导航（已完成）

- [x] path 类型 tag，支持虚拟文件夹结构
- [x] 主界面左侧目录栏显隐（`f`）
- [x] 文件夹式主机列表导航
- [x] `j` 目录跳转浮层
- [x] 带 `path:` 前缀的目录范围搜索

## v0.6 — 安全存储

- [x] 系统钥匙串凭证存储（登录密码 + 私钥口令）
- [x] 认证成功后静默保存到系统钥匙串
- [x] Linux 无 Secret Service 时禁止保存，仅允许临时输入并提示安装能力

## v0.7 — 完善文件操作

- [ ] 文件夹递归传输
- [x] 远程文件编辑：下载 → 默认图形化应用打开 → 监听变更 → 自动上传
- [ ] 双面板模式（本地 + 远程同屏）
- [ ] 文件多选批量操作（删除、传输）
- [ ] 传输队列 + 后台传输
- [ ] 断点续传

## v0.8 — 高级网络功能

- [ ] SSH 端口转发管理（本地 / 远程 / 动态）
