# 设计文档：v0.4 使用体验增强

## 功能范围

- **连接历史**：记录每台主机的最近连接时间，影响主机列表排序（无搜索时按时间倒序，有搜索时叠加近期加权）
- **连通性检测**：选中主机悬停 1 秒后自动发起异步 TCP probe，结果以彩色圆点显示，光标移走即清除

---

## 子系统一：连接历史

### 存储

新增文件 `~/.config/sush/history.toml`，格式：

```toml
[connections]
"prod-web-01" = "2026-04-24T10:23:41Z"
"prod-db-01"  = "2026-04-23T08:15:00Z"
```

key 为 `host.id`，value 为 RFC3339 格式的最近一次成功连接时间戳。只保留最新一次，不做历史列表。

文件路径：`config_dir().join("history.toml")`，与 `hosts.toml` 同目录。

### 新模块 `src/config/history.rs`

```rust
pub struct ConnectionHistory {
    // host_id -> last connected (UTC)
    connections: HashMap<String, DateTime<Utc>>,
    path: PathBuf,
}

impl ConnectionHistory {
    pub fn load(path: PathBuf) -> Self { ... }
    pub fn record(&mut self, host_id: &str) { ... }  // 更新内存 + 写文件
    pub fn last_connected(&self, host_id: &str) -> Option<&DateTime<Utc>> { ... }
}
```

`record` 在成功建立连接后调用：
- SSH 模式：`ssh_connect_and_takeover` 成功打开 PTY 后
- SFTP 模式：`sftp_connect` 成功后

### App 新增字段

```rust
pub connection_history: ConnectionHistory,
```

启动时 `ConnectionHistory::load(config_dir().join("history.toml"))` 初始化。

### 排序逻辑

`apply_search()` 统一处理两种情况：

**无搜索（`search_query` 为空）**：
```
sorted by: last_connected DESC (None 排最后), hostname ASC (同级二排)
```

**有搜索**：
```rust
recent_bonus(host_id) = match days_since_last_connected {
    0..=7   => 30,
    8..=30  => 15,
    31..    => 5,
    None    => 0,
};
combined_score = fuzzy_score + recent_bonus
sorted by: combined_score DESC
```

模糊分数仍是主导，近期加权只在相近匹配间起提升作用。

## 子系统二：连通性检测

### App 新增字段

```rust
pub hover_host_id: Option<String>,
pub hover_since: Option<Instant>,
pub probe_rx: Option<tokio::sync::oneshot::Receiver<bool>>,
pub probe_result: Option<bool>,
```

### 状态机

**光标移动时**（`select_next`/`select_previous`/`apply_search` 改变选中项，以及从 Edit/Import 模式返回 Main 时）：

```rust
fn on_selection_changed(&mut self) {
    let new_id = self.selected_host_id();
    if self.hover_host_id != new_id {
        self.hover_host_id = new_id;
        self.hover_since = Some(Instant::now());
        self.probe_rx = None;
        self.probe_result = None;
    }
}
```

**每次 Tick**（`handle_tick`）：

```rust
// 1. 收取已完成的 probe 结果
if let Some(rx) = &mut self.probe_rx {
    if let Ok(result) = rx.try_recv() {
        self.probe_result = Some(result);
        self.probe_rx = None;
    }
}

// 2. 检查是否满 1 秒且未发起 probe
if self.probe_rx.is_none() && self.probe_result.is_none() {
    if let Some(since) = self.hover_since {
        if since.elapsed() >= Duration::from_secs(1) {
            if let Some(host) = self.selected_host() {
                let hostname = host.hostname.clone();
                let port = host.port;
                let (tx, rx) = tokio::sync::oneshot::channel();
                tokio::spawn(async move {
                    let result = probe(hostname, port).await;
                    let _ = tx.send(result);
                });
                self.probe_rx = Some(rx);
            }
        }
    }
}
```

**TCP probe 函数**（放在 `app.rs` 或 `src/ssh/probe.rs`）：

```rust
async fn probe(hostname: String, port: u16) -> bool {
    tokio::time::timeout(
        Duration::from_secs(3),
        tokio::net::TcpStream::connect((hostname.as_str(), port)),
    )
    .await
    .map(|r| r.is_ok())
    .unwrap_or(false)
}
```

### 渲染

`HostList` widget 新增参数 `probe: Option<Option<bool>>`：

| 值 | 含义 | 圆点颜色 |
|----|------|----------|
| `None` | 未悬停或刚切换，不显示圆点 | — |
| `Some(None)` | probing 中（已悬停 ≥1s，等待结果）| 灰色 |
| `Some(Some(true))` | 可达 | 绿色 |
| `Some(Some(false))` | 不可达 | 红色 |

**左侧缩进修正**：
- `highlight_symbol` 从 `"● "` 改为 `"  ● "`
- 非选中行前缀从 `"  "` 改为 `"    "`

选中行的圆点颜色由 `probe` 参数决定，非选中行无圆点（保持空格占位）。

`main_view::render` 向 `HostList` 传入：
```rust
let probe = if app.probe_result.is_some() {
    Some(app.probe_result)
} else if app.probe_rx.is_some() {
    Some(None) // probing
} else {
    None
};
```
